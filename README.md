## Installatie

Deze app is ontwikkeld met:
* Express
* Node.js
* EJS
* MongoDB

Clone de repository
```
git clone https://github.com/Davi9898/bRAIN.git
```

Installeer alle packages
```
npm install
```

Start de Applicatie
```
node start
```
